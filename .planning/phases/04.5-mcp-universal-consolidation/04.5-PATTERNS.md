### Shared-C: Per-request error handling (express middleware)

**Source:** `src/mcp-server.ts:65-86`, `src/mcp-stream-server.ts:265-276`.

**Apply to:** session-mode handler in `src/mcp-stream-server.ts` (after rewrite).

**Pattern (typed-error → status-code map):**
```typescript
} catch (err) {
  if (err instanceof UnknownToolError) { res.status(404).json({ error: 'unknown_tool', tool_name }); return; }
  if (err instanceof BadRequestError) { res.status(400).json({ error: 'bad_request', field: err.field }); return; }
```
