### QUAL-02 aggregate shape

```
scripts/phase5-qual02-aggregate.ts reads:
  input: list of call_ids (via argv or stdin)
  source: ~/nanoclaw/voice-container/runs/turns-*.jsonl (Phase 2 VOICE-10 JSONL format)
  filter: entries where call_id ∈ input
  metric: latency_ms = T4 - T0 per turn (TTS-first-byte - VAD-end)

Output JSON:
  {
    call_count: number,
    turn_count: number,
    p50_ms: number,
    p95_ms: number,
    threshold_p50_ms: 900,
    threshold_p95_ms: 1500,
    passes: boolean
  }

Exit 0 if passes, 1 if not. Used in 05-GATE-EVIDENCE.md verification.
```
