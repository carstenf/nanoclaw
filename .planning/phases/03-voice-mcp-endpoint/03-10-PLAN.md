<!-- andy-agent-runner.ts public API -->
```typescript
export interface AndyVoiceResult {
  voice_short: string       // max 3 sentences, post-parsed
  discord_long: string | null
  container_latency_ms: number
}

export interface AndyRunnerDeps {
  runContainer?: typeof runContainerAgent
  loadMainGroup?: () => RegisteredGroup | null
  loadSkill?: (topic: string) => Promise<{exists: boolean, body: string | null, path: string}>
  now?: () => number
  timeoutMs?: number
}

export async function runAndyForVoice(
  request: string,
  deps?: AndyRunnerDeps,
): Promise<AndyVoiceResult>
```

Internal flow:
1. Load skill body via skill-loader (topic='andy' ‚Üí `data/skills/ask-core-andy/SKILL.md`).
2. If no skill exists: return voice_short="Andy-Skill ist nicht konfiguriert.", discord_long=null.
3. Fetch main-group from DB (is_main=1 row). If none: error fallback.
4. Build container-input: prompt = skill_body + "\n\n=== REQUEST ===\n" + request, chatJid=main.jid, groupFolder="main", isMain=true, sessionId=uuid.
5. Race: runContainerAgent(main, input, noopOnProcess, noopOnOutput) vs timeout (30s).
6. Parse stdout: find last `{...}`-block that JSON-parses. Extract voice_short + discord_long.
7. Validate voice_short: ‚â§3 sentence-ends (`/\.|\!|\?/g` count ‚â§3). Truncate if longer.
8. Return {voice_short, discord_long, container_latency_ms}.

```markdown
# ask-core-andy skill

Du bist NanoClaw-Andy im Voice-Channel-Kontext. Carsten hat dich waehrend eines Telefonats via "frage Andy"-Tool angesprochen.

## Deine Rolle
- Du hast Zugriff zu deinem vollen Tool-Set (WebSearch, WebFetch, Browser, alle MCP-Tools) und deinem Context (CLAUDE.md, Hindsight, Memory).

Deine letzte Nachricht MUSS ein JSON-Block sein, EXAKT in diesem Format:
```
